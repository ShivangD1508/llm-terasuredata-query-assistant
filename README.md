# LLM Treasure data Query Assistant

Natural language interface that converts business questions into SQL queries for Treasure data and returns structured outputs.

## Project Overview

This project allows users to ask questions in natural language and generates Treasure data-compatible SQL using an LLM-based workflow.

## Features

* Natural language to SQL generation
* Treasure data query execution
* Interactive query handling
* Structured result output

## Tech Stack

* Python
* SQL
* Treasure data
* LLM Integration

## Use Cases

* Business query automation
* Faster analytical exploration
* SQL assistance for non-technical users

## Future Improvements

* Streamlit interface
* Query validation layer
* Role-based schema selection
